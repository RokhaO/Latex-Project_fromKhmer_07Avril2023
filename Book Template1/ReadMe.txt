0.) You are recommended to instal the "latest version" of TeX Distribution like MiKTeX v2.9 (Windows), TeXLive 2015 (Windows or Ubuntu), or MacTeX (Mac OSX)

1.) You must have fonts "Khmer OS" and "Khmer M1" installed in your operating system.

2.) Try to run "test.tex" with "XeLaTeX" engine first. If it produce output without error continue to stage 3.

3.) You must run "khmerbook.tex" with "XeLaTeX" engine.

Note: The default engine you are using to typeset English document is "pdfLaTeX". That default engine cannot be use to typeset Khmer document at the present time.
